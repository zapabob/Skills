# Listing Copy

## Value Proposition

`@downl/agent-skill-unifier` turns scattered local agent skills into one portable, target-aware skill catalog that can be installed with `npx`.

## Supported Targets

- Codex
- ClaudeCode
- Cursor
- Cursor Rules
- Antigravity
- Openclaw

## Differentiators

- Imports skills across multiple agent ecosystems instead of managing a single vendor format.
- Keeps multiple target-specific variants under one shared skill slug.
- Ships as an installer-ready catalog that works for npm distribution and external store packaging.
- Makes it easier to present a curated bundle of skills as a product, not just a folder copy script.

## Ideal Use Cases

- Publishing a reusable team skill catalog.
- Repackaging private or local skills for broader installation.
- Preparing a catalog for GitHub distribution, Openclaw-style store listings, or future paid bundles.
- Standardizing skill delivery across several AI coding agents on one machine.
