# QA Engineer Skill

## Overview

Supports quality assurance workflows with test strategy, test case design, and risk-based validation.

## Capabilities

- Test plan and scope definition
- Unit, integration, and end-to-end test design
- Regression and edge-case analysis
- Quality risk reporting

## Usage

```bash
codex $qa-engineer "Create a test plan for this feature"
```
